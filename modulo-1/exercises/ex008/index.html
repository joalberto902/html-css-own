Principais formatações:

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>(Não semântica)

Nesta frase temos um termo em destaque usando a tag <strong>(semântica)

Itálico / Ênfase

Nesta frase, tenho um termo em itálico usando a tag <i>(Não semântica)

Nesta frase, tenho um termo enfatizado usando a tag <em>(semântica)

Texto Marcado

Podemos criar uma marcação usando a tag <mark>

E no outro parágrafo temos outro texto marcado de outra cor.

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído usando a tag <del> para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins>

É possível usar a tag <u>, mas ela já está caindo em desuso

Texto sobrescrito

Para inserir coisas como x20+3

Texto subscrito

Para inserir coisas como H2O
